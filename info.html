What is machine learning?

Machine learning is the study of computer algorithms that can improve automatically through experience and by the use of data. It is seen as a part of artificial intelligence. Machine learning algorithms build a model based on sample data, known as training data, in order to make predictions or decisions without being explicitly programmed to do so. This came to existence after a long search for artificial intelligence. People wanted to find a easier way to program machines and find algorithms and that's why they developed machine learning.

Examples of machine learning

Learning algorithms are used in a wide variety of applications such as

  • Medicine
  • Email filtering
  • Speech recognition
  • AI chess bots
  • Relevant digital advertisements

How does it work?

The computer basically identifies patterns in data and learns from each mistake it makes. There are three broad categories of machine learning shown beneath with a description of what it is.

Supervised learning: The machine is provided with a lot of information about a certain case and the case outcome is also provided. The outcome is the labeled data and the rest of the information is used as input. For example, a machine is shown 100 cases when the customers defaulted on a loan and 100 cases where customers didn't default. Things like age and salary are the input, and default/ not-default is the labeled data. With this information the machine can learn the relationships between the outcome default and the input information.

Unsupervised learning: The computer must learn without the help from the user. The machine identifies patterns in the data that is not obvious for the human eye. It recognises patterns and is quite useful to help us make decisions. For example: if we did not know which customers defaulted on loans, but gave the borrower information to the machine, it would be able to see patterns among the borrowers and group them. Unsuperivised learning is used for the detections like uncovering fraudulent transactions or payments.

Reinforcement learning: This is the closest to humans when it comes to the way of learning. The algorithm continually learns from its environment by interacting with it. For each action it gets a positive or a negative reward. For example bankloans. A reinforcement learning algorithm looks at the information and classifies the custumer as a high-risk customer. When the person defaults, the algorithm gets a positive reward. If the person doesn't default, it gets a negative reward. The rewards help the agent understand the environment and the problem better and they can now learn for future cases.

Machine learning versus traditional programming

Traditional programming takes out the data, uses the rules as input and applies the rules to the data to get the answers. Whereas in Machine Learning (ML) the data and the answers are used for input and the new learned rules come as output. It teaches the computer new rules in a complex space, a space harder to comprehend by humans. For example, a traditional computer program for activity recognition for someone's speed (data) and definiton of activity (rules). The problem: different people walk with variable speeds depending upon, for example, age. With ML lots of example activities with their labels will be given and it will then learn the rules to predict future activity. Traditional programming is manually done and if something occurs, like an error, humans would have to fix it. Machine learning on the other hand learns from its own mistakes and can fix those errors without the need for human assistance.
